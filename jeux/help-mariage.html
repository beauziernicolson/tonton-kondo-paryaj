Mariage — règles officielles

Le Mariage est désormais calculé à partir des 3 lots Borlette du tirage publié. L’ordre des numéros n’a pas d’importance : 12-45 est identique à 45-12.

← Retour à la page Mariage

Qu’est-ce que Mariage ?

Vous jouez deux numéros sur un tirage. Pour gagner, les deux numéros doivent être présents parmi les 3 lots officiels Borlette du même tirage.

  • Vous choisissez un tirage.
  • Vous choisissez deux numéros de 00 à 99.
  • Le ticket est vérifié automatiquement quand un résultat Borlette est publié.

Comment jouer

  • Choisissez un tirage Borlette.
  • Sélectionnez un premier et un deuxième numéro.
  • Indiquez votre mise.
  • Validez votre ticket ; le wallet est débité comme d’habitude.
Gain potentiel = mise × 5000

Règle de validation

Les deux numéros joués doivent être présents dans les 3 lots du tirage :

  • 1er lot
  • 2e lot
  • 3e lot

Si les deux numéros figurent dans cette liste, le ticket est gagnant. L’ordre n’a pas d’importance.

Exemples

Résultat Borlette du tirage :

1er lot = 45 2e lot = 12 3e lot = 89
  • 45 × 12 → gagnant
  • 12 × 89 → gagnant
  • 45 × 77 → perdant

Le gain est de mise × 5000.
