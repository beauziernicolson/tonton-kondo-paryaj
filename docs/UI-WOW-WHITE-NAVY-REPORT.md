# Tonton Kondo — WOW White / Navy / Yellow Redesign

## Direction artistique
- White-first interface with navy structural anchors and precise yellow accents.
- Editorial hierarchy, generous whitespace and asymmetric visual rhythm.
- Reduced neon, excessive gradients and repetitive rounded cards.
- Unified form, table, navigation, game-card, auth and dashboard styling.

## Scope
- Visual stylesheet: `assets/tk-wow-redesign.css`
- Injected into all active HTML pages.
- Backup mockups `dashboard.backup.html` and `dashboard2.html` intentionally untouched.

## Preserved
- JavaScript logic
- Supabase queries and RPCs
- SQL files
- IDs, links, form names and routes
- Games and calculations

## Responsive targets
- 320 / 360 / 390 / 430 px
- 768 / 1024 / 1366 px
- 1500 px+ displays
